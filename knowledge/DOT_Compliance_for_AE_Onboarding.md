## Driver Qualification File (DQF)

A DOT-mandated digital folder that proves a driver is eligible to operate a CMV.

| Component | Description |
|---|---|
| **DOT-Compliant Application** | Must include 10-year job history (for driving roles), explanation of gaps >1 month, traffic violations, accident history, license info, and signed certification. |
| **Motor Vehicle Record (MVR)** | Pulled from all states the driver held a license in the past 3 years. Must be reviewed at hire and annually. |
| **Safety Performance History (SPH)** | Verification of previous DOT-regulated employers (past 3 years), covering dates of employment, accidents, and drug/alcohol testing violations. |
| **Medical Examiner's Certificate** | Valid medical card from a certified examiner. Must be renewed every 2 years or more frequently if required. |
| **Driver's License** | Must be valid and appropriate class (CDL or not). |
| **Road Test Certificate or CDL** | If no CDL, employer must administer and retain a road test certificate. |
| **Annual Review + Certificate of Violations** | Annually document MVR review and get a signed list of traffic violations. |
| **ELDT Certificate (if applicable)** | For drivers who got a CDL after Feb 2022. |

> ✅ **Foley automates collection, reminders, and digital storage for every item above.**

## Drug & Alcohol Testing (DAT)

A federally mandated testing program for all CDL drivers operating safety-sensitive vehicles.

| Requirement | Description |
|---|---|
| **Pre-employment drug test** | Required before driver can perform safety-sensitive functions. |
| **Random testing** | 50% of drivers/year for drugs, 10% for alcohol. Must be scientifically random and unannounced. |
| **Other tests** | Post-accident, Reasonable Suspicion, Return-to-Duty, Follow-up. |
| **MRO & DER** | Results must be reviewed by a Medical Review Officer. Employers must have a Designated Employer Representative. |
| **Retention Requirements** | Records must be kept 1–5 years depending on type. |

## FMCSA Clearinghouse

A centralized database that tracks CDL drivers' drug & alcohol program violations and test history.

| Concept | Description |
|---|---|
| **Full Query** | Required before hiring a CDL driver. Driver must consent through their Clearinghouse account. |
| **Limited Query** | Required annually for all active CDL drivers. Consent can be bulk/long-term. |
| **Reporting** | Employers must report positive test results, refusals, and Return-to-Duty completions. |
| **Consequences** | Drivers with unresolved violations cannot perform safety-sensitive functions. |
| **Fines** | Up to $5,833 per driver for failing to run queries. |

## Understanding Regulatory Thresholds

### CDL-Regulated Drivers — Full DOT Compliance Required
**Vehicle Weight: Over 26,001 lbs**

Requires:
- Driver Qualification File
- Drug & Alcohol Testing Program
- Clearinghouse Queries
- Valid CDL License

### Non-CDL but DOT-Regulated Drivers
**Vehicle Weight: 10,001 – 26,000 lbs**

Requires:
- Driver Qualification File
- Valid Standard License
- Medical Certificate
- ❌ No drug testing or Clearinghouse required

### Non-Regulated Drivers
**Vehicle Weight: Under 10,001 lbs**

Requires:
- ❌ No formal DOT compliance
- (Optional) MVR checks for insurance or internal policy
